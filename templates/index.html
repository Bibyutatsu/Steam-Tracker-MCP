Connecting to Steam Intelligence

A step-by-step guide to integrate your specialized Steam MCP server into modern AI environments.

!

Authentication Required

This server is protected for your security. You must append your MCP_TOKEN (configured in your Space Secrets) to your connection URLs using the ?token= parameter.

Steam Intelligence is a Model Context Protocol (MCP) server that provides real-time access to Steam library data, pricing, and social activity. Choose your platform below to begin setup.

🚀 Host Your Own Instance

The easiest way to use this MCP server is to host your own private instance on Hugging Face Spaces:

  1. Duplicate the Space: Click the "Duplicate Space" button in the Hugging Face menu (top right).
  2. Configure Secrets: Go to your new Space's Settings > Variables and secrets and add:
    • STEAM_WEB_API_KEY: Your key from Steam Dev Portal.
    • STEAM_ID: Your 64-bit Steam ID.
    • MCP_TOKEN: Your secret token (e.g., the one you use in ?token=...).
  3. Deploy: Hit Save. Your private MCP server will be live in seconds.

Claude Desktop

The standard way to connect to any SSE-enabled MCP server. Add the following entry to your configuration file:

{
  "mcpServers": {
    "steam-intelligence": {
      "url": "https://bibyutatsu-steam-tracker-mcp.hf.space/sse?token=YOUR_TOKEN"
    }
  }
}

Cursor / Windsurf

For IDEs that support direct URL connection strings, use the optimized integration path:

https://bibyutatsu-steam-tracker-mcp.hf.space/mcp?token=YOUR_TOKEN
